## ‚ú® Key Platform Features

* **Course Enrollment & Management:** Users can easily browse, enroll in, and manage their Software Engineering courses.
* **Themed Interface:** Supports both **Dark and Light Mode** for reduced eye strain during long study sessions.
* **Progress Tracking:** Interactive dashboards allow students to monitor their completion status and learning milestones.
* **Fully Responsive Design:** Ensures a flawless learning experience on any device, from desktop to mobile.
* **Secure User Accounts:** User registration and data are secured and managed using **Firebase Authentication**.
* **Smooth Interaction:** Features an engaging user interface powered by **Framer Motion for modern animations**.
* **Optimized Performance:** Utilizes dynamic content loading for faster course access and responsiveness.
